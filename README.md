# [Project Euler](https://projecteuler.net/about)

**Project Euler** (<a>https://projecteuler.net/about</a>)
is a super cool website that I got to know a few days ago.
Actually it was introduced to me by a friend of mine.
I gave it a check and found out that it hosts some
mathematical problems which are unsolvable by only human
or using only calculators. These problems are designed to
be solved by using computer programs and they are open to
use any algorithms you can come up with.

But the website only accepts the final answers and do
not provide a feature that saves the code written by the
programmer. So I decided to host the programs/algorithms
that I write to solve a problem in GitHub.

## Repo organization

The repo will contain folders named by the problem serial
number, and it will contain the problem explainations, used
approaches etc. and the program itself.

Feel free to give feedback on my algorithms or discuss on a
specific topic based on the used approach, or give any
suggestions to improve my program.

You can find me in [Project Euler](https://projecteuler.net/about)
website as **`galaxy38`** :>
